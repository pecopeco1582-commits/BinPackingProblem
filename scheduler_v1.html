About
A quantum-inspired bin-packing scheduler that assigns cloud jobs to servers using a MILP optimiser. Formulates the problem as a Quadratic Program (binary variables, linear constraints) and solves for the minimum number of active servers.

Server Count
Each server has 2 vCPUs, 4 GB RAM, 40 GB NVMe and 10 Gbps bandwidth. The solver finds the minimum number of servers needed.
3
Binary Variables
—
Variables = (jobs × servers) + servers. The MILP solver handles up to 100 vars without any degradation.
This tool demonstrates a quantum-inspired optimisation approach to cloud resource scheduling. It formulates job-to-server assignment as a binary optimisation problem and solves it using scipy.milp (exact MILP), falling back to numpy brute-force or a classical greedy scheduler if needed. The objective is to minimise active servers while respecting CPU, RAM, NVMe and bandwidth constraints.
